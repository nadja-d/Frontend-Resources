Front-end Website Resources

w3schools Link

W3Schools is a freemium educational website for learning coding online. Initially released in 1998, it derives its name from the World Wide Web but is not affiliated with the W3 Consortium. W3Schools offers courses covering all aspects of web development. W3Schools also publishes free HTML templates.

CodeHS Link

CodeHS is an interactive online learning platform offering computer science and programming instruction for schools and individual learners.

CSS-Tricks Link

Daily articles and tips about CSS, HTML, JavaScript, and all things related to web design and development.

MDN Web Docs Link

MDN Web Docs, previously Mozilla Developer Network and formerly Mozilla Developer Center, is a documentation repository and learning resource for web developers.

Flexbox Froggy Link

Flexbox Froggy is a game for learning CSS flexbox.

JavaScript for Cats Link

JavaScript for Cats is a website - introduction to JavaScript for new programmers.

Interneting is Hard Link

Interneting is Hard makes it easier by putting together a comprehensive set of web development tutorials to help transform complete beginners into talented Interneting professionals.

Learn-JS Link

learn-js.org is a free interactive JavaScript tutorial for people who want to learn JavaScript.
